Task 3

It is not wise to name DF as df because df is an existing function. Naming an object with the same name would overwrite existing objects from installed packages.

It’s okay to use the names of existing objects for list element, column, or row names, however. They don’t interact.

# Create data frame
DF <- data.frame(v1, v2, v3, v4)
names(DF) <- c("a", "b", "c", "grp")
str(DF)
#> 'data.frame':    20 obs. of  4 variables:
#>  $ a  : int  67 38 0 33 86 42 13 81 58 50 ...
#>  $ b  : int  9 4 7 1 2 7 2 3 1 5 ...
#>  $ c  : num  0.266 0.372 0.573 0.908 0.202 ...
#>  $ grp: chr  "I" "D" "G" "A" ...

If DF$grp is a factor vector. You can change it to a character either by specifying stringsAsFactors = FALSE in the call to data.frame, or by coercing like this:

DF$grp <- as.character(DF$grp)
